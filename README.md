# Business Listings Dashboard

A full-stack data dashboard built with React, FastAPI, and MySQL.

## Tech Stack
- Frontend: React.js with Recharts
- Backend: FastAPI (Python)
- Database: MySQL

## Features
- 250+ business listings across 10 cities
- City-wise business count (Bar Chart)
- Category-wise distribution (Pie Chart)
- Source-wise distribution (Pie Chart)
- Bulk insert API endpoint
- REST APIs for all dashboard data

## How to Run

### Backend

cd frontend
npm install
npm start

### Frontend

cd frontend
npm install
npm start

## API Endpoints
- GET /city-wise — City wise business count
- GET /category-wise — Category wise count
- GET /source-wise — Source wise count
- POST /insert-listings — Bulk insert listings
